THIS NOTE CONTAINS MEANING TO THE DOCUMENT SENT

1. fake_news_bilstm_model.h5 : is the model itself
2. fon_dataset : contains dataset in only fon language
3. LSTM : documentation to the project
4. multilingual_news_dataset : contains dataset used to train the model
5. News Detector : this is an ipynb file and it contains the codes used to build the model
6. news_pred : this contains the code used to build the streamlit app and the app can be tested by typing "streamlit run news_pred.py"
7. tokenizer.pickle : it is used to convert text data to numbers, and it is needed to help the model understand the input
8. french_dataset : contains dataset in only french language
9. yoruba_dataset : contains dataset in only yoruba language
